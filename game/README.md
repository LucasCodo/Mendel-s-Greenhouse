# Mendel's Greenhouse Game Package

This is the Pyxel game package for Mendel's Greenhouse.

Run from this directory:

```powershell
poetry install
poetry run mendels-greenhouse
```

Development checks:

```powershell
poetry run poe check
```

The repository root remains reserved for high-level documentation, specs, and
project governance. A future NiceGUI site package can be added beside this
package without mixing game runtime code with web account/save management.
