# Pyxel Assets

Production art assets for Mendel's Greenhouse live in this package directory.

The primary native Pyxel resource file is expected at:

```text
game/mendels_greenhouse/assets/mendels_greenhouse.pyxres
```

Early prototypes may use code-generated placeholder graphics until the first
`.pyxres` file is created.

The current `.pyxres` file is generated by:

```powershell
cd game
poetry run poe build-assets
```

It contains:

- Four Mendel Pea pod sprites for the MVP phenotype combinations.
- 64 x 64 top-navigation and resource icons.
- Initial sound effects for selection, breeding, reveal, completion, invalid
  actions, and conveyor/music loops.

HUD frames, panels, button surfaces, and conveyor panel shapes are generated by
runtime drawing code instead of being packed into `.pyxres`.

Do not edit generated resource positions without updating
`tools/build_assets.py` and the runtime code that references sprite
coordinates.

## Custom Font

The project font is generated as:

```text
game/mendels_greenhouse/assets/mendel_5x7.bdf
```

Pyxel loads it with `pyxel.Font(...)` for title and HUD display text. It is an
ASCII-oriented MVP display font; localization-specific glyph coverage remains
future work.

## Background Image

The current runtime greenhouse background is stored as:

```text
game/mendels_greenhouse/assets/greenhouse_background_640x360.png
```

The runtime loads the 640 x 360 PNG into a custom `pyxel.Image` and draws it
behind the gameplay UI. Temporary previews and source images should not be
committed; keep the repository limited to runtime assets and reproducible
generation scripts.
