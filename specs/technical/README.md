# Technical Direction

This directory records approved implementation-level direction for Mendel's Greenhouse.

## Documents

- [pyxel.md](pyxel.md) - official engine and pixel-art constraints.
- [platform-roadmap.md](platform-roadmap.md) - web target and future NiceGUI scope.
- [project-structure.md](project-structure.md) - Poetry flat-layout package structure and asset location.
- [localization.md](localization.md) - initial English and Brazilian Portuguese support.
- [testing.md](testing.md) - pytest-based automated testing strategy.
- [future-platform.md](future-platform.md) - future save, packaging, web delivery, architecture, and NiceGUI decisions.
- [development-tooling.md](development-tooling.md) - dev dependencies, Poe tasks, Ruff, Babel automation, and refactoring policy.
- [implementation-readiness.md](implementation-readiness.md) - resolved readiness decisions for starting the MVP.

## Boundaries

The engine, long-term web direction, initial Poetry package layout, future save format, future packaging path, and future NiceGUI account model are selected. Hosting, database technology, deployment workflow, and final NiceGUI routing are not finalized yet. Do not invent those details without updating this directory and the related specs.
