# Technical Direction

This directory records approved implementation-level direction for Mendel's Greenhouse.

## Documents

- [pyxel.md](pyxel.md) - official engine and pixel-art constraints.
- [platform-roadmap.md](platform-roadmap.md) - web target and future NiceGUI scope.
- [project-structure.md](project-structure.md) - Poetry flat-layout package structure and asset location.
- [localization.md](localization.md) - initial English and Brazilian Portuguese support.
- [testing.md](testing.md) - pytest-based automated testing strategy.

## Boundaries

The engine, long-term web direction, and initial Poetry package layout are selected. Detailed architecture, persistence, packaging, and hosting are not finalized yet. Do not invent those details without updating this directory and the related specs.
