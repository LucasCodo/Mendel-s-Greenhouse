# Learning Objectives

## Purpose

This document maps genetics concepts to the systems that teach them.

## Dominance

How it appears: dominant alleles determine visible traits when present.

When it appears: analyzer level 1.

System that teaches it: phenotypic observation and phenotypic contracts.

## Recessiveness

How it appears: recessive traits appear when both alleles are recessive.

When it appears: analyzer level 1.

System that teaches it: repeated observation of offspring traits.

## Genotype

How it appears: complete allele representation such as `AaBb`.

When it appears: analyzer level 2.

System that teaches it: genetic sequencing and genotypic contracts.

## Allele

How it appears: each gene is represented by two alleles, such as `A` and `a`,
with one inherited from each parent.

When it appears: analyzer level 2.

System that teaches it: genetic analyzer allele breakdown, genotype inspection,
and knowledge tree explanations.

## Phenotype

How it appears: visible plant traits such as red flowers or broad leaves.

When it appears: from the start.

System that teaches it: plant observation, contracts, and collection.

## Homozygosity

How it appears: matching allele pairs such as `AA` or `aa`.

When it appears: analyzer level 2.

System that teaches it: genotype inspection and exact-genotype contracts.

## Heterozygosity

How it appears: mixed allele pairs such as `Aa`.

When it appears: analyzer level 2.

System that teaches it: genotype inspection and hybrid preservation.

## Allele Segregation

How it appears: each parent contributes one allele per gene to each offspring.

When it appears: reinforced from level 2 onward.

System that teaches it: crossbreeding results and genotype comparison.

## Independent Assortment

How it appears: multiple genes combine independently across offspring.

When it appears: analyzer level 3 and multi-gene species.

System that teaches it: probability predictions, multi-gene contracts, Punnett
square summaries, and the knowledge tree.

Key conditions:

- The traits are controlled by different genes.
- The genes assort independently.
- Each parent contributes one allele per gene.
- The expected `9:3:3:1` phenotype ratio applies to a complete-dominance
  dihybrid cross such as `AaBb x AaBb`.

## 9:3:3:1 Dihybrid Ratio

How it appears: the analyzer groups the expected phenotypes of a valid
independent dihybrid cross into four outcome classes with a `9:3:3:1` ratio.

When it appears: analyzer level 3 after the player can inspect probabilities for
two independently assorting genes.

System that teaches it: probabilistic analyzer summaries, Scientific Customer
contracts, breeding history, and the knowledge tree.

## Genetic Probability

How it appears: expected distributions for selected parent pairs.

When it appears: analyzer level 3.

System that teaches it: probabilistic analysis and probabilistic contracts.

## Genetic Planning

How it appears: the player compares possible parent pairs and plans cross paths.

When it appears: analyzer level 4.

System that teaches it: genetic simulator and complex Corn, Tomato, and Orchid goals.
