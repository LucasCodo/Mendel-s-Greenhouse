# Game Overlays

## Purpose

Overlays provide focused feedback without replacing the current gameplay context.

## New Plant Discovery

Trigger:

- First discovery of species, phenotype, or genotype.

Content:

- Plant visual.
- Discovery type.
- Name or label.
- Collection destination.
- Reward if any.

## Contract Completed

Trigger:

- Contract objective reaches completion.

Content:

- Contract title.
- Customer type.
- Reward.
- Any unlocks or milestones.

## Species Unlock

Trigger:

- Player purchases or unlocks a new species.

Content:

- Species name or placeholder.
- Gene count.
- New complexity introduced.

## Analyzer Level Up

Trigger:

- Player buys analyzer upgrade.

Content:

- New analyzer level.
- New information unlocked.
- New contract type if applicable.

## Reward Received

Trigger:

- Contract, discovery, or milestone reward.

Content:

- Credit amount.
- Reward source.
- Updated total.
