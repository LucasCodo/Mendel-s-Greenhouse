# Tutorial Screen

## Purpose

The Tutorial Screen provides onboarding with minimal text and visual examples.

## Topics

- Genes.
- Alleles.
- Genotypes.
- Phenotypes.
- Crossbreeding.
- Contracts.
- Greenhouse storage.

## Tutorial Style

- Prefer visual examples over definitions.
- Use one concept per step.
- Keep text short.
- Let the player act quickly.
- Repeat core patterns through gameplay.

## Layout

```text
+------------------------------------------------------------+
| Tutorial: Phenotypes                                       |
+---------------------------+--------------------------------+
| Visual Example            | Short Explanation              |
| Red flower vs white flower | Traits are visible outcomes.   |
+---------------------------+--------------------------------+
| [Back]                                      [Try It] [Next] |
+------------------------------------------------------------+
```

## Example Sequence

1. Identify visible traits.
2. Select two parent plants.
3. Start a cross.
4. Watch offspring appear.
5. Deliver matching plant to a simple contract.
6. Open analyzer level 1.

## Rules

- Do not expose advanced probability before the player reaches analyzer level 3.
- Do not require reading long paragraphs.
- Tutorial examples must use official content from [../../content/content-bible.md](../../content/content-bible.md).
