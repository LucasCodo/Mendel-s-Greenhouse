# Layout Sketches

Place generated layout reference images for Mendel's Greenhouse in this
directory.

Use these sketches as visual references for Pyxel layout implementation. Keep
final gameplay rules, screen responsibilities, and UI requirements documented in
the relevant files under `specs/ui/`.

For scene-specific layout decomposition, use the Obsidian-friendly scene wiki in
`specs/ui/scenes/`. The current main scene reference belongs under
`specs/ui/scenes/main-game/references/`.
