These images are voting references for the Mendel Guide character direction.
They are not approved production sprites yet.

Use them to choose the strongest silhouette, costume language, and personality
before creating final Pyxel-ready portrait and icon assets.

## Evaluation Criteria

- Reads clearly as a friendly tutorial mentor.
- Looks like retro pixel art, not a smooth illustration.
- Fits the cozy scientific greenhouse tone.
- Can be simplified into `64 x 64` or `96 x 96` Pyxel portraits.
- Does not imply fantasy powers or non-Mendelian mechanics.
