# Entry Detail Panel

![[specs/ui/scenes/collection/references/components/collection-component-entry-detail-panel.png]]

## Purpose

Shows full details for the selected discovered collection entry.

## Elements

- Entry title.
- Close/return button when required by layout.
- Large preview.
- Trait summary.
- Information section.

## Visibility

Respect analyzer level. Hidden information remains locked until the relevant
system reveals it.
