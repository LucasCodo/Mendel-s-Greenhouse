# Collection Tabs

![[specs/ui/scenes/collection/references/components/collection-component-collection-tabs.png]]

## Purpose

Switches between collection categories.

## Tabs

- Species.
- Phenotypes.
- Genotypes.
- Specimens when implemented.

## Behavior

The active tab uses green highlight. Inactive tabs use dark wood panels.
