# Collection Entry Grid

![[specs/ui/scenes/collection/references/components/collection-component-collection-entry-grid.png]]

## Purpose

Displays discovered and undiscovered records in a stable card grid.

## Behavior

- Clicking a card selects the entry.
- Discovered entries show art, name, and discovered status.
- Locked entries show silhouette, lock, and broad placeholder only.
- Selection updates the detail panel.
