# Garden Slot Grid

![[specs/ui/scenes/garden/references/components/garden-component-garden-slot-grid.png]]

## Purpose

Displays stored plants, empty slots, and locked capacity in a scannable grid.

## Layout

- 5 columns.
- 3 visible rows in the reference.
- Wood frame with darker slot cards.

## Behavior

- Clicking a slot selects it.
- Selected slot receives a bright outline.
- Empty/locked slots show a muted plus marker.
- Occupied slots show plant art and optional count/generation badge.
