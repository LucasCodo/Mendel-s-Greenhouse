# Selected Plant Preview

![[specs/ui/scenes/garden/references/components/garden-component-selected-plant-preview.png]]

## Purpose

Shows a large readable version of the selected plant.

## Visual Direction

Use a framed green display area with subtle sparkles or highlights. The preview
must make phenotype traits legible without relying on text.
