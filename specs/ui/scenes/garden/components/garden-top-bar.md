# Garden Top Bar

![[specs/ui/scenes/garden/references/components/garden-component-garden-top-bar.png]]

## Purpose

Shows persistent brand and resource state while the player manages stored
plants.

## Elements

- Logo plaque.
- Credits counter.
- Garden capacity counter.

## Visual Direction

Use the same wood-frame and dark-counter treatment as the main scene so the
Garden feels like part of the same interface family.
