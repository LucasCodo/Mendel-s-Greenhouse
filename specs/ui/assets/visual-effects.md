# Visual Effects

## Purpose

Visual effects provide readable feedback for discovery, rarity, success, warnings, and errors.

## Discovery Glow

Usage:

- New phenotype.
- New genotype.
- New species.

Direction:

- Soft gold or seed-yellow glow.
- Should not hide plant traits.

## Particles

Usage:

- Discovery.
- Contract completion.
- Species unlock.

Direction:

- Botanical particles such as pollen, leaf motes, or small sparkles.
- Short duration.
- Low density.

## Rarity Effects

| Rarity | Effect |
| ------ | ------ |
| Common | No special effect |
| Uncommon | subtle green edge |
| Rare | blue shimmer |
| Very Rare | lavender-gold highlight |

## Positive Feedback

Usage:

- Valid delivery.
- Contract progress.
- Successful purchase.

Direction:

- Green check.
- Progress pulse.
- Short confirmation flash.

## Negative Feedback

Usage:

- Invalid delivery.
- Not enough credits.
- Locked requirement.

Direction:

- Red or amber shake/highlight.
- Clear explanation text.
- Do not rely only on color.

## Contract Match Effect

Usage:

- Offspring matches active contract.

Direction:

- Contract panel pulse.
- Matching trait badge highlight.
- Progress bar increment.

## Rules

- Effects must preserve readability.
- Effects must be brief.
- Effects should reinforce existing systems, not imply new mechanics.
